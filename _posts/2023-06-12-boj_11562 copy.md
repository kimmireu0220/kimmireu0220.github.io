## 문제

[백준 11562번: 카드 (실버 4)](https://www.acmicpc.net/problem/11562)

## 풀이 아이디어

- 카드가 주어질 때마다 딕셔너리에 {카드 번호: 카드 개수} 형식으로 추가
- 카드 개수 오름차순, 카드 번호 내림차순 정렬
- 마지막 요소의 카드 번호 출력

## 정답 코드

```python
import sys
input = sys.stdin.readline

n = int(input())
dict = {}

for _ in range(n):
    num = int(input())
    if dict.get(num):
        dict[num] += 1
    else:
        dict[num] = 1

a = sorted(dict.items(), key=lambda x: (x[1], -x[0]))
print(a[-1][0])
```
