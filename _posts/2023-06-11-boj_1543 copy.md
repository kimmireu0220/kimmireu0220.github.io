## 문제

[백준 1543번: 문서 검색 (실버 4)](https://www.acmicpc.net/problem/1543)

## 풀이 아이디어

- 문서의 끝에 도달할 때까지 인덱스를 증가시킨다. (0 ~ 문서의 길이)
- 단어가 등장할 때마다 단어의 길이만큼 인덱스 증가 및 카운트 1 증가시킨다.

## 정답 코드

```python
import sys
input = sys.stdin.readline

a = input().rstrip()
b = input().rstrip()
result = 0
idx = 0

while idx < len(a):
    if b == a[idx : idx + len(b)]:
        result += 1
        idx += len(b)
    else:
        idx += 1

print(result)
```
