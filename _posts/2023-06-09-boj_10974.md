## 문제

[백준 10974번: 모든 순열 (실버 3)](https://www.acmicpc.net/problem/10974)

## 풀이 아이디어

- 백트래킹

## 정답 코드

```python
import sys
input = sys.stdin.readline

def recursive(n):
    if n == N:
        print(*result)
        return

    for i in range(1, N + 1):
        if not visited[i]:
            visited[i] = True
            result.append(i)
            recursive(n + 1)
            visited[i] = False
            result.pop()

N = int(input())
result = []
visited = [False] * (N + 1)

recursive(0)
```
